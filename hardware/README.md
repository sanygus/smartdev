##Hardware
####Потребуется
* Raspberry Pi Model B+ V 1.2
* Camera module for Raspberry Pi
* Arduino UNO Rev.3
* Электромагнитное реле (hrs4h-s-dc5v)
* Блок питания 5V 2A
* Некоторые другие радиодетали:
  * Breadboard Half+
  * Резисторы (220Ом * 6, 10кОм * 2)
  * Потенциометр
  * Кнопка
  * Светодиоды * 3
  * Транзистор полевой MOSFET
  * Много проводов :)
